# 🛒 MERN E-Commerce Website

A full-stack e-commerce web application built using the MERN stack that provides a seamless online shopping experience. Users can browse products, search for items, view product details, add products to the cart, and place orders through a responsive and user-friendly interface.

## 🚀 Features

* User Authentication
* Browse Products
* Product Search
* Product Details Page
* Shopping Cart Management
* Order Placement
* Responsive Design
* RESTful API Integration
* MongoDB Database

## 🛠️ Tech Stack

### Frontend

* React.js
* React Router
* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose
